Atividade - projeto MultiSEL para o AMUSE VR

Usando o conteúdo aprendido no tutorial, crie um projeto MultiSEL para o AMUSE VR. Para auxilia-lo, será disponibilizado um editor de código com um escopo inicial de um projeto MultiSEL que pode ser utilizado como base para o início de seu projeto, além disto, mais abaixo está sendo disponibilizado um rascunho de um projeto completo, similar ao do tutorial para sua consulta.

O tema do seu projeto será: "Crie uma apresentação interativa mulsemídia 360°, sobre a divulgação turística da cidade do Rio de Janeiro".

Para a criação dessa apresentação, segue a lista mínima de tarefas a serem realizadas em seu projeto MultiSEL:

  • crie um vídeo ou imagem 360° na primeira cena;
  • crie um vídeo ou imagem 360° na segunda cena;
  • crie na primeira cena uma imagem ou vídeo 2D;
  • relacione o início da imagem ou vídeo 2D com o vídeo ou imagem em 360°, na primeira cena;
  • torne interativo o vídeo ou imagem 2D de forma que possa iniciar a cena 2;
  • na cena 2, crie algum efeito sensorial qualquer; e
  • para finalizar, salve o projeto um arquivo ou copie o projeto da apresentação, juntamente com o tempo da tarefa e envie para o endereço de e-mail informado nesta página.

Os dados utilizados em seu projeto como dados pessoais ou endereço dos arquivos (atributos src), devem ser todos fictícios, a fim de não identificar o participante.

Anote o tempo de total, que levará para criar o código MultiSEL (somente o tempo para conclusão da atividade prática).

Para criação do código, você pode optar por usar o editor de texto fornecido abaixo (já com rascunho inicial) ou algum de sua preferência.

O código deve ser enviado em arquivo, através dos formatos xml, txt ou doc, para o e-mail: flavio.farias@midiacom.uff.br, use no título do e-mail a palavra "experimento" e no corpo informar o seu nome e duração da atividade.
