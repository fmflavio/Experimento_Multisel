Atividade - projeto MultiSEL para o AMUSE VR




Primeiramente, para iniciarmos esta atividade, é necessário que você registre o seu tempo de início desta atividade e o tempo do término para posteriormente ser usado em formulário.

Usando o conteúdo aprendido no tutorial, crie um projeto MultiSEL para o AMUSE VR.

O tema do seu projeto será: "Crie uma apresentação interativa mulsemídia 360°, sobre a divulgação turística da cidade do Rio de Janeiro".

O projeto deverá conter duas cenas com vídeo 360°. Na primeira cena, deve ser incluída uma imagem 2D exibida 1 (um) segundo após a cena ser iniciada. Esta imagem deve ser interativa, ou seja, ao ser clicada pelo usuário, deve iniciar a segunda cena. Na segunda cena, deve ser incluído um efeito sensorial de vento (WindType) com duração de 3 segundos.

O nome dos arquivos que podem ser utilizados no seu projeto são esses:

  • Vídeos em 360 graus para background
    • cristo360.mp4
    • paodeacucar360.mp4
    • carnaval360.mp4
    • copacabana360.mp4
    • futebol360.mp4
  • Imagens em 2D
    • cristo2d.jpg
    • paodeacucar2d.jpg
    • carnaval2d.jpg
    • copacabana2d.jpg
    • futebol2d.jpg

Dica: lembre-se de criar uma porta na primeira cena para refenciar a imagem 2D ao criar o relacionamento interativo entre as duas cenas. Caso precise relembrar de algo, clique aqui para abrir o tutorial MultiSEL novamente.

Para auxiliá-lo, será disponibilizado uma pasta para seu projeto que conterá um código XML com a estrutura básica de um projeto MultiSEL (chamado Projeto_MultiSEL_for_AMUSEVR.xml) e um esquema para linguagem (chamado MultiSEL_Schema_to_AMUSEVR.xsd). Além disto, no fim desta página é disponibilizado um exemplo de um projeto completo, similar ao do tutorial para sua consulta.

Para baixar o projeto básico clique aqui (é recomendavel o uso do esquema MultiSEL disponibilizado, o mesmo serve para validar o código em tempo de execução e antes do envio).

Caso saiba como utilizar um esquema XML, fique a vontade para utilizar qualquer ferramenta de edição de texto de sua preferência, caso não saiba/entenda, siga este tutorial aqui.

Para finalizar, salve o projeto (aquivo em XML), pois será necessário fazer upload posteriormente no formulário da página inicial. E lembre-se de registrar o tempo que levou para criar/codificar o projeto MultiSEL.



Terminou o projeto, salvou o arquivo XML? Então vamos retornar para pagina principal e completar a Fase 3 do experimento. Clique aqui. para ser redirecionado para o questionário sobre esta atividade.
