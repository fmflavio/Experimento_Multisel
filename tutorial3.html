Tutorial - Mídias e Efeitos Sensoriais (ES) em um projeto MultiSel


A imagem a baixo mostra exemplos de descrever mídias e ES.



As tags media e effect são usadas para representar mídias e efeitos sensoriais respectivamente.

Tanto as mídias quando os ES, possuem atributos id como identificador único e type para definir o seu tipo, as mídias também possuem o atributo src que define o endereço do arquivo.

As mídias podem ser dos tipos:

  • video - para vídeo
  • image - para imagens e fotos
  • audio - para áudios
  • text - para mensagens de texto
  • ico - para botões interativos

Os Efeitos Sensoriais podem ser dos tipos:

  • LightType - para efeito de luz
  • SteamType - para efeito de vapor
  • WindType - para efeito de vento

As tags property, são usadas para definir as características da mídia ou efeito sensorial.

Todas as tags property possuem atributos name que define o nome da características e a tag value que define o conteúdo.

As mídias do tipo vídeo e imagem podem ser apresentadas de diversas formas quando apresentada ou não uma property diferenciadora, a seguir os tipos apresentações possíveis dessas mídias:

  • Vídeo em 360° - para exibir o vídeo ao fundo em 360°, basta acrescentar a tag , como na linha 3
  • Vídeo em Picture in Picture - para exibir o vídeo em PIP, basta acrescentar a tag
  • Imagem em 360° - para exibir a imagem em 360°, basta acrescentar a tag

Para todos os outros casos de mídia e ES, deve-se omitir as tags citadas e descrever a posição no cenário 3D com os valores x, y e z, como nas linhas 10, 11 e 12

Todas as outras tags exibidas no exemplo são opcionais, porem recomendadas
