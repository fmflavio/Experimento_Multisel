Tutorial - Mídias e Efeitos Sensoriais (ES) em um projeto MultiSEL


A imagem a baixo mostra exemplos de como se escreve a mídias e ES.



As tags media e effect são usadas para representar mídias e efeitos sensoriais (ES) respectivamente.

Tanto as mídias quando os ES, possuem atributos id como identificador único e type(tipo) para definir o tipo de mídia, as mídias também possuem o atributo src que define o endereço do arquivo.

As mídias podem ser dos tipos:

  • video - para vídeo;
  • image - para imagens e fotos;
  • audio - para áudios;
  • text - para mensagens de texto; e
  • ico - para botões interativos.

Os Efeitos Sensoriais podem ser dos tipos:

  • WindType - para efeito de vento;
  • HeatType - para efeito de calor;
  • ScentType - para efeito de odores;
  • LightType - para efeito de luz; e
  • SteamType - para efeito de vapor.

As tags property, são usadas para definir as características da mídia ou efeito sensorial.

Todas as tags property possuem atributos name que define o nome das propriedades e a tag value que define o conteúdo.

As mídias do tipo vídeo e imagem podem ser apresentadas de diversas formas. Para diferenciar a apresentação, existe uma tags property própria para essa finalidade. A seguir os tipos apresentações possíveis dessas mídias:

  • Vídeo em 360° - para exibir o vídeo ao fundo em 360°, basta acrescentar a tag property , como na linha 3.
  • Vídeo em Picture in Picture (PIP) - para exibir o vídeo em PIP, basta acrescentar a tag property com o atributo name=”pip” e value=”true”.
  • Imagem em 360° - para exibir a imagem em 360°, basta acrescentar a tag property , como na linha 3.

Para todos os outros casos de mídia e ES, deve-se omitir as tags citadas e descrever a posição no cenário 3D com os valores x, y e z em valores reais, como nas linhas 10, 11 e 12 do exemplo. As demais tags exibidas nos exemplos como duration, volume e loop são opcionais, porem recomendadas.
